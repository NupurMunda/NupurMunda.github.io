Image in Words

datascience
Image in Words

Framework: Google Colaboratory(Ipython)

Technologies: Convolutional neural network, Recurrent Neural Network, Deep Learning, Python, Keras, Pandas

Synopsis: The input to the model is an image, and the output is a sentence describing the image content. This model generates captions from a fixed vocabulary. The model consists of an encoder model – a deep convolutional net using the VGG16 architecture trained on ImageNet data – and a decoder model – an LSTM network that is trained conditioned on the encoding from the image encoder model.

Dataset: Flickr 8k dataset, This dataset contains almost 8000 images along with their description.
